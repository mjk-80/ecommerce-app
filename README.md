# اپلیکیشن فروشگاهی (E-commerce App)

این یک اپلیکیشن فروشگاهی ساده است که با استفاده از **React**، **TypeScript** و **Vite** ساخته شده است. این پروژه برای نمایش نحوه‌ی ساخت یک سایت فروشگاهی پاسخگو (Responsive) و کاربردی با استفاده از تکنولوژی‌های مدرن فرانت‌اند طراحی شده است.

## راه‌اندازی پروژه

برای شروع کار با این پروژه، مراحل زیر را دنبال کنید:

### 1. کلون کردن مخزن
پروژه را به ماشین محلی خود کلون کنید:

```bash
git clone https://github.com/username/my-ecommerce-app.git
```

### 2.نصب وابستگی ها

```bash
cd my-ecommerce-app
npm install
```
### 3.اجرای پروژه

```bash
npm run dev
```
حال پروژه شما در آدرس http://localhost:3000 در مرورگر قابل مشاهده خواهد بود.


### ساختار پروژه:
در اینجا یک نمای کلی از ساختار پوشه‌های پروژه آورده شده است:
```bash
my-ecommerce-app/
├── public/                 # منابع عمومی مانند تصاویر
├── src/                    # کد منبع اپلیکیشن
│   ├── components/         # کامپوننت‌های قابل استفاده مجدد مانند Header و Footer
│   ├── pages/              # صفحات مختلف مانند Home و Product Details
│   ├── styles/             # استایل‌های عمومی و استایل‌های اختصاصی هر کامپوننت
│   ├── App.tsx             # کامپوننت اصلی اپلیکیشن
│   ├── main.tsx            # نقطه ورود اپلیکیشن
│   └── router.tsx          # مدیریت مسیریابی اپلیکیشن
├── package.json            # اطلاعات پروژه و وابستگی‌ها
└── tsconfig.json           # فایل تنظیمات TypeScript
```
